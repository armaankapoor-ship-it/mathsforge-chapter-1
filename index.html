01Relations

Cartesian product and ordered pairs

A x B is the set of all ordered pairs (a,b) where a is in A and b is in B.

  • CBSE writing: Write A x B explicitly using braces when sets are small.
  • JEE focus: Count pairs using n(A x B) = n(A)n(B).
  • Common trap: (a,b) is not the same as (b,a) unless a = b.
02Relations

Relation as a subset of A x B

A relation R from A to B is any subset of A x B.

  • CBSE writing: State R subset A x B before listing ordered pairs.
  • JEE focus: Number of relations from A to B is 2^(mn).
  • Common trap: A relation need not connect every element of A.
03Properties

Reflexive relation

A relation on A is reflexive if every a in A has (a,a) in R.

  • CBSE writing: Check all diagonal pairs.
  • JEE focus: For n elements, every reflexive relation must contain n fixed diagonal pairs.
  • Common trap: One missing diagonal pair destroys reflexivity.
04Properties

Symmetric relation

R is symmetric if (a,b) in R implies (b,a) in R.

  • CBSE writing: Pair every non-diagonal ordered pair with its reverse.
  • JEE focus: Use reverse-pair check for counterexamples.
  • Common trap: Symmetric does not mean every pair is present.
05Properties

Transitive relation

R is transitive if (a,b) and (b,c) imply (a,c).

  • CBSE writing: Search chains of length two.
  • JEE focus: A single broken chain is enough to disprove transitivity.
  • Common trap: Do not check only adjacent numbers; check ordered-pair chains.
06Properties

Equivalence relation

An equivalence relation is reflexive, symmetric, and transitive together.

  • CBSE writing: Prove the three properties under separate headings.
  • JEE focus: Equivalence classes partition the set.
  • Common trap: Two out of three properties are not enough.
07Functions

Function and mapping

A function from A to B assigns every element of A exactly one image in B.

  • CBSE writing: Mention exactly one image for each domain element.
  • JEE focus: Vertical-line idea becomes unique-image test in sets.
  • Common trap: One input with two images means not a function.
08Functions

Domain, codomain and range

Domain is input set, codomain is declared target set, range is actual set of images.

  • CBSE writing: Write range separately after applying the rule.
  • JEE focus: Onto depends on codomain, not just expression.
  • Common trap: Range and codomain are often different.
09Functions

One-one function

A function is one-one if distinct inputs have distinct images.

  • CBSE writing: Use f(x1)=f(x2) implies x1=x2.
  • JEE focus: Graphically, horizontal line cuts at most once.
  • Common trap: Many-one functions fail one-one even if they are onto.
10Functions

Onto function

A function is onto if every element of codomain has a preimage.

  • CBSE writing: Take arbitrary y in codomain and find x in domain.
  • JEE focus: Compare range with codomain.
  • Common trap: A function can be one-one but not onto.
11Functions

Bijective function

Bijective means both one-one and onto.

  • CBSE writing: Conclude bijective only after proving both.
  • JEE focus: A function has inverse iff it is bijective.
  • Common trap: Invertible is not guaranteed by one-one alone unless codomain matches range.
12Operations

Composition of functions

(g o f)(x) means g(f(x)); f acts first, g acts second.

  • CBSE writing: Check range of f lies in domain of g.
  • JEE focus: Composition is generally not commutative.
  • Common trap: g o f is not the same as f o g.
13Operations

Invertible functions

A function f is invertible if there exists f^-1 such that f^-1 o f = identity and f o f^-1 = identity.

  • CBSE writing: State bijection condition before finding inverse.
  • JEE focus: Solve y=f(x) for x and interchange symbols.
  • Common trap: Inverse relation may fail to be a function if f is not one-one.
14Binary operations

Binary operation

A binary operation * on A maps every ordered pair of A x A to a unique element of A.

  • CBSE writing: First check closure.
  • JEE focus: Use operation tables for finite sets.
  • Common trap: If output leaves A, it is not a binary operation on A.
15Binary operations

Identity and inverse elements

Identity e satisfies a*e=e*a=a. Inverse b of a satisfies a*b=b*a=e.

  • CBSE writing: Identity must work from both sides.
  • JEE focus: Left identity alone may not be enough in non-standard operations.
  • Common trap: Different elements can fail to have inverses even when identity exists.

Counting

Number of ordered pairs

n(A x B) = n(A) n(B)

Cartesian product size multiplies the set sizes.

  • Example: If |A|=3 and |B|=4, then |A x B|=12.
  • Trap: This counts all possible pairs, not a chosen relation.
Counting

Number of relations

Number of relations from A to B = 2^(mn)

Every subset of A x B is a relation.

  • Example: If |A|=2, |B|=3, total relations = 2^6=64.
  • Trap: Do not use mn; that counts pairs only.
Reflexive

Reflexive condition

For every a in A, (a,a) in R

Every element must relate to itself.

  • Example: On {1,2}, R must contain (1,1) and (2,2).
  • Trap: Extra pairs do not compensate for a missing diagonal pair.
Symmetric

Symmetric condition

(a,b) in R => (b,a) in R

Every arrow must have its reverse.

  • Example: (2,3) requires (3,2).
  • Trap: Diagonal pairs do not need a separate reverse.
Transitive

Transitive condition

(a,b),(b,c) in R => (a,c) in R

Chains must close.

  • Example: (1,2),(2,3) require (1,3).
  • Trap: Check all possible chains.
Function

One-one test

f(x1)=f(x2) => x1=x2

Equal outputs force equal inputs.

  • Example: f(x)=2x+3 is one-one over R.
  • Trap: Do not prove x1=x2 => f(x1)=f(x2); that is ordinary function behaviour.
Onto

Onto test

Range(f) = Codomain

Every codomain element is hit.

  • Example: f:R->R, f(x)=x^3 is onto.
  • Trap: Changing codomain can change onto status.
Composition

Composition

(g o f)(x) = g(f(x))

Apply f first, then g.

  • Example: If f(x)=x+1 and g(x)=x^2, then (g o f)(x)=(x+1)^2.
  • Trap: Order matters.
Inverse

Inverse function

f^-1 o f = I_A and f o f^-1 = I_B

Inverse reverses the mapping exactly.

  • Example: f(x)=x+5 has f^-1(x)=x-5.
  • Trap: Inverse exists as a function only for bijections.
Binary

Closure

a,b in A => a*b in A

Operation output remains inside the set.

  • Example: Addition modulo 3 is closed on {0,1,2}.
  • Trap: Ordinary subtraction is not closed on natural numbers.
Binary

Commutative

a*b = b*a

Order does not affect result.

  • Example: Addition is commutative.
  • Trap: Matrix multiplication style operations often fail commutativity.
Binary

Associative

(a*b)*c = a*(b*c)

Grouping does not affect result.

  • Example: Addition of integers is associative.
  • Trap: Do not assume every custom operation is associative.

Derivation bank

Board-ready derivations and proofs.

Each proof is broken into steps so students can reproduce it in exams.

Prove number of relations is 2^(mn) 2^(mn)
  1. A relation from A to B is any subset of A x B.
  2. If |A|=m and |B|=n, then |A x B|=mn.
  3. A set with mn elements has 2^(mn) subsets.
  4. Therefore number of relations from A to B is 2^(mn).
  • Shortcut: Count pairs first, then count subsets.
  • Warning: Do not confuse number of pairs with number of relations.
Standard proof of one-one using algebra f(x1)=f(x2) => x1=x2
  1. Assume f(x1)=f(x2).
  2. Substitute the formula for f.
  3. Simplify both sides using valid algebra.
  4. If the result is x1=x2, f is one-one.
  • Shortcut: Start from equal outputs.
  • Warning: Never start by assuming x1=x2 for one-one proof.
Onto proof template Range = codomain
  1. Let y be any element of the codomain.
  2. Solve y=f(x) for x.
  3. Show that the obtained x belongs to the domain.
  4. Conclude every y has a preimage, so f is onto.
  • Shortcut: Arbitrary y method.
  • Warning: Check the domain condition for x.
Inverse from bijection f^-1 exists iff f is bijective
  1. One-one ensures no output is linked to two inputs.
  2. Onto ensures every codomain element is used.
  3. Together, each codomain element has exactly one preimage.
  4. So the reverse mapping is also a function.
  • Shortcut: Exactly one preimage is the whole idea.
  • Warning: If either condition fails, inverse function fails.
Identity element test a*e=e*a=a
  1. Choose a possible identity element e.
  2. Check a*e=a for every a in the set.
  3. Check e*a=a for every a in the set.
  4. Only then call e the identity.
  • Shortcut: Check both sides.
  • Warning: Left identity only is not enough.

Practice zone

Original practice questions with solutions.

Use the filter to isolate CBSE, JEE, proof, graph, or trap-based questions.

CBSEEasy

1. If A={1,2} and B={a,b,c}, find n(A x B).

Answer: 6

n(A x B)=2*3=6.

Trap: Do not list only matching pairs.

CBSEEasy

2. Define a relation from A to B.

Answer: A subset of A x B.

Any selected set of ordered pairs from A x B is a relation.

Trap: A relation need not be a function.

CBSEMedium

3. Check whether R={(1,1),(2,2),(1,2),(2,1)} on {1,2} is an equivalence relation.

Answer: Yes.

It is reflexive, symmetric, and transitive; in fact it is the universal relation on {1,2}.

Trap: Do not forget transitivity.

JEEMedium

4. How many relations are possible from a 3-element set to a 2-element set?

Answer: 64

A x B has 6 elements, so number of subsets is 2^6=64.

Trap: Answer 6 counts pairs, not relations.

FunctionMedium

5. Is f:R->R, f(x)=x^2 one-one?

Answer: No.

f(1)=f(-1)=1 but 1 != -1.

Trap: Positive-looking formula can still be many-one.

FunctionMedium

6. Is f:R->R, f(x)=x^3 onto?

Answer: Yes.

For any y in R, choose x=cube root y.

Trap: Do not restrict x to positive values.

CompositionMedium

7. If f(x)=x+2 and g(x)=3x, find (g o f)(x).

Answer: 3x+6

g(f(x))=g(x+2)=3(x+2)=3x+6.

Trap: Do not compute f(g(x)) unless asked.

InverseMedium

8. Find inverse of f(x)=2x-5 over R.

Answer: f^-1(x)=(x+5)/2

Let y=2x-5, solve x=(y+5)/2, then replace y by x.

Trap: Check that f is bijective over R.

BinaryMedium

9. Is a*b=a+b+1 closed on integers?

Answer: Yes.

For integers a,b, a+b+1 is also an integer.

Trap: Closure depends on the set.

BinaryHard

10. For a*b=a+b+ab on R, find identity.

Answer: 0

a*e=a+e+ae=a gives e(1+a)=0 for all a, so e=0. Also e*a=a.

Trap: Identity must work for every a.

ProofHard

11. Prove f:R->R, f(x)=5x-7 is one-one and onto.

Answer: Bijective.

Equal outputs give x1=x2. For y in R, x=(y+7)/5 is real, so onto.

Trap: Both one-one and onto are needed.

TrapMedium

12. Can a function be onto but not one-one?

Answer: Yes.

Example f:{1,2,3}->{a,b}, f(1)=a, f(2)=a, f(3)=b is onto but not one-one.

Trap: Onto only checks whether codomain is fully hit.

Must-know formulas

  • Number of ordered pairs: n(A x B) = n(A) n(B)
  • Number of relations: Number of relations from A to B = 2^(mn)
  • Reflexive condition: For every a in A, (a,a) in R
  • Symmetric condition: (a,b) in R => (b,a) in R
  • Transitive condition: (a,b),(b,c) in R => (a,c) in R
  • One-one test: f(x1)=f(x2) => x1=x2
  • Onto test: Range(f) = Codomain
  • Composition: (g o f)(x) = g(f(x))
  • Inverse function: f^-1 o f = I_A and f o f^-1 = I_B
  • Closure: a,b in A => a*b in A

Must-avoid traps

  • (a,b) and (b,a) are different ordered pairs.
  • A relation may not be a function.
  • Every domain element must have exactly one image.
  • Onto depends on codomain.
  • g o f means f first, then g.
  • Inverse function needs bijection.
  • Closure must be checked before other binary operation properties.
